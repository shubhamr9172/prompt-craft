# PromptCraft Academy

A world-class, interactive learning platform for mastering prompt engineering.

## How to Run
1.  Simply open `index.html` in any modern web browser (Chrome, Firefox, Edge).
2.  No server or installation required!

## Features
-   **Interactive Modules**: 8 comprehensive lessons on prompt engineering.
-   **AI Playground**: Simulate AI interactions and get instant feedback on your prompts.
-   **Quizzes**: Test your knowledge and track your progress.
-   **Dark/Light Mode**: Toggle between themes for comfortable reading.
-   **Responsive Design**: Works on mobile, tablet, and desktop.

## Tech Stack
-   HTML5
-   CSS3 (Variables, Flexbox, Grid, Glassmorphism)
-   Vanilla JavaScript (ES6+)
